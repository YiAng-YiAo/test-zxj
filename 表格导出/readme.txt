导出：
1使用前先修改路径配置文件，配置里面的两个目录变量，默认为上级目录output下
2将要导出的表拖到导出.bat上 导出单个表
3直接执行导出.bat 导出全部表

导入：
1将要导入的lua文件或文件夹拖到导入.bat上，在lua文件所在文件夹会生成表格
2导入只支持与导出相对应的格式，即单个文件形式为xxConfig={ {id=1,name=value},{id=2,name=value},...}
	目录形式时子文件名为id.后缀，子文件内容为{name=value,name=value}
3生成的表格需要自行添加描述字段，特殊文本可能需要手动增加双引号

4遇到乱码报错的用tool下的转换工具转换下utf8无bom格式
